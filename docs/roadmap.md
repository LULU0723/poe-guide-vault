# Roadmap

## 優先事項

- [ ] 匯入時提供「新增」、「覆蓋同 ID」與「取代全部」三種模式
- [ ] JSON Schema 驗證與清楚的錯誤訊息
- [ ] 攻略刪除、複製及重新命名
- [ ] 技能組新增、刪除與排序
- [ ] 裝備列刪除與排序
- [ ] 自動壓縮貼入圖片，並顯示本機儲存空間用量
- [ ] 攻略內容與遊玩核取進度分開匯出

## 後續構想

- [ ] 支援多角色／多聯盟進度
- [ ] 版本差異與攻略更新紀錄
- [ ] PoEDB、交易市集與 pobb.in 快速連結
- [ ] 列印／PDF 專用排版
- [ ] PWA 離線安裝
- [ ] 從標準 JSON 產生巴哈文章格式

Roadmap 會依實際遊玩與維護需求調整，不代表功能承諾或固定時程。
