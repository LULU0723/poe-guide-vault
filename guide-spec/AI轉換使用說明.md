# POE 攻略庫 AI 轉換規格包｜使用說明

## 建議的 ChatGPT 專案配置

建立一個新的 ChatGPT 專案，例如「POE 攻略轉換器」，並進行以下設定：

1. 把 `POE攻略庫_AI轉換專案指令.md` 貼到專案指令。
2. 把 `POE攻略庫_JSON格式規格.json` 與 `POE攻略庫_空白匯入範本.json` 上傳為專案檔案。
3. 每次新增對話時，提供攻略網址、PDF、PoB、影片字幕或自己的筆記。
4. 要求 AI：「整理成流亡攻略庫可匯入 JSON」。
5. 將 AI 輸出的 JSON 程式碼另存成 `.json` 檔。
6. 在流亡攻略庫點擊「匯入」，選擇該檔案。

## 更新既有攻略

目前網站的匯入功能會以匯入檔內的攻略清單取代瀏覽器現有清單。因此在匯入新資料前，建議先按「匯出」備份目前攻略庫。

如果只想新增一個流派，最安全的流程是：

1. 先從網站匯出目前完整攻略庫。
2. 把匯出的 JSON 與新攻略資料一起交給 AI。
3. 要求 AI 保留原本 `guides` 內容，並在陣列最後新增一份攻略。
4. 將合併後 JSON 匯入網站。

## 圖片處理

AI 輸出的 JSON 不應包含 Base64 圖片。匯入攻略後，開啟編輯模式，在「個人筆記與圖片」欄位直接貼上剪貼簿圖片即可。

圖片會存在該瀏覽器的本機儲存空間。大量或高解析圖片可能超過瀏覽器容量，重要圖片建議保留原始檔。

## 備份原則

- 大幅修改前先匯出一次。
- 換電腦或瀏覽器前先匯出。
- JSON 可以放在 Google Drive、GitHub 私人儲存庫或自己的備份資料夾。
- 不要只依賴瀏覽器本機資料。
